# Звуковые эффекты (SFX)

Здесь лежат файлы, которые подключает игра. **Лицензия MIT в корне репозитория относится к коду**, а не автоматически к этим аудиофайлам. Для каждого трека нужно иметь право на использование (своя запись, лицензия с банка звуков, договор и т.д.).

## Список файлов и заметки по репозиторию

| Файл | Назначение в коде | Техн. заметка |
|------|-------------------|---------------|
| `ui.mp3` | UI | Базовый эффект |
| `move.mp3` | Ход | |
| `attack.mp3` | Атака | |
| `hit.mp3` | Получение урона | |
| `win.mp3` | Резерв / не основной путь победы | В `game.js` победа идёт через `the-sound-of-victory-winning.mp3` |
| `death.mp3` | Резерв | В `game.js` поражение идёт через `you-died-45f.mp3` |
| `transition.mp3` | Переход этажа | **Совпадает по содержимому с `ui.mp3`** (заглушка, одинаковый SHA256) |
| `the-sound-of-victory-winning.mp3` | Победа | Источник: каталог [Zvukogram — звуки выигрыша](https://zvukogram.com/category/zvuki-vyiigryisha/). Уточните на сайте условия для вашего сценария (игра, распространение). |
| `you-died-45f.mp3` | Поражение | Источник: каталог [VoiceBot — звуки проигрыша (Game Over)](https://voicebot.su/ru/category/proigrysh/). На странице раздела указано ограничение на использование; **перечитайте актуальный текст** и пользовательское соглашение на сайте. |

## Что заполнить владельцу проекта

Скопируйте таблицу и допишите для своих файлов:

| Файл | Источник (сайт / своё) | Лицензия / основание | Дата добавления |
|------|-------------------------|----------------------|-----------------|
| `the-sound-of-victory-winning.mp3` | [Zvukogram, раздел «Звуки выигрыша»](https://zvukogram.com/category/zvuki-vyiigryisha/) | Сверьте с актуальным пользовательским соглашением и блоком «Правообладателям» на сайте Zvukogram (ссылки в подвале страниц). | |

Для игры с публичным распространением это может быть недостаточно — сверьтесь с актуальным пользовательским соглашением VoiceBot (подвал сайта) и при сомнениях замените трек. | |
| Остальные | | | |

Для `the-sound-of-victory-winning.mp3` источник зафиксирован; **итоговое право на встраивание в игру и публикацию** всё равно нужно сверить с актуальными правилами Zvukogram (они могут меняться).

Для `you-died-45f.mp3` источник зафиксирован; из‑за возможного ограничения «информационные и учебные цели» этот файл **особенно важно** сверить с правилами VoiceBot или заменить на трек с явной лицензией под ваш сценарий.
